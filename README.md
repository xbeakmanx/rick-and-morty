## Installation

---

A little intro about the installation. Also you can watch the project in production here: https://rick-and-morty-two-dun.vercel.app/

```
$ git clone https://github.com/xbeakmanx/rick-and-morty.git
$ cd rick-and-morty
$ npm install
$ npm start
```
